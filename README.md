# Genius
O jogo "Genius" é um classico jogo da memória baseado em cores. Neste repositório contém a minha versão do jogo em formato de site.

## Linguagens Utilizadas

- HTML
- CSS
- JAVA SCRIPT

## Ferramentas Utilizadas

- VScode
- Github Pages

## Como jogar

- O jogo consiste em repetir uma sequência de cores que aparece na tela.            
- Conforme você acerta as sequências, mais você avança na rodada.            
- Ao passar de rodada, o jogo se torna mais difícil.
- Você só ganha quando passa por todas as rodadas.            
- Ao errar a sequência, o jogo termina e você pode tentar novamente.
